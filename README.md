# OmniContent AI Engine 🚀

Este es tu sistema automático para generar dinero creando contenido con IA.

## Requisitos
1. **Node.js** instalado.
2. **Gemini API Key**: Consíguela gratis en [Google AI Studio](https://aistudio.google.com/app/apikey).

## Configuración
1. Abre el archivo `backend/.env`.
2. Reemplaza `YOUR_API_KEY_HERE` con tu llave de API de Gemini.

## Cómo Ejecutar
1. **Iniciar el Backend**:
   - Abre una terminal en `backend`.
   - Ejecuta: `node server.js`
2. **Iniciar el Dashboard**:
   - Abre otra terminal en `frontend`.
   - Ejecuta: `npm run dev`
3. Abre el enlace que te dé la terminal (usualmente `http://localhost:5173`).

## Estrategia de Monetización
- Selecciona temas en tendencia en el dashboard.
- Genera contenido con el botón "Generate Content".
- Copia el contenido generado y publícalo en redes sociales (Twitter, Instagram, Facebook).
- **Importante**: Incluye enlaces de afiliados de productos relacionados (puedes usar Amazon Associates, por ejemplo).
